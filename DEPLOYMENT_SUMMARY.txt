================================================================================
🆘 TROUBLESHOOTING
================================================================================

Frontend Issues:
  Problem: Network errors
  Solution: Check NEXT_PUBLIC_API_URL in Vercel dashboard

Backend Issues:
  Problem: 500 errors
  Solution: Check logs in Railway/Render dashboard
           Verify environment variables are set

CORS Issues:
  Problem: CORS errors in console
  Solution: Update allow_origins in backend/app/main.py
           Add your Vercel URL to CORS whitelist

Database Issues:
  Problem: Connection timeout
  Solution: Verify DATABASE_URL is correct
           Check PostgreSQL is running

For detailed troubleshooting, see DEPLOYMENT.md
